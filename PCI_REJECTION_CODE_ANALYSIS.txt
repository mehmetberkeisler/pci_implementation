THE PCI ALGORITHM PIPELINE:
─────────────────────────────────────────────────────────────────────────────
1. Load epochs around stimulus markers
2. Z-score each channel against its baseline activity
3. Bootstrap null distribution (preserving spatial correlation)
4. Create binary significance matrix via hypothesis testing
5. Calculate Lempel-Ziv complexity and entropy
6. Apply quality gates (entropy threshold)
7. Return PCI or 0 if quality gates fail

The entropy calculation (lines 64-92):
```python
def compute_entropy(ss_matrix):
    p1 = float(np.sum(ss_matrix)) / ss_matrix.size  # fraction of 1s
    
    if p1 == 0.0 or p1 == 1.0:
        return 0.0, p1  # H = 0 if all 0s or all 1s
    
    H = -p1 * np.log2(p1) - (1.0 - p1) * np.log2(1.0 - p1)
    return H, p1
```
